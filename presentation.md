### Why Git
<div style="position: absolute; bottom: 0px; right: 0">
"There is no way to do CVS right"
<br>
<em>- Linus Torvalds</em>
</div>
- Distributed Version Control
    - Less Trust Required
    - Offline Productivity
    - Fast
- Branches are awesome
- Big Development Momentum
    - Hosting
    - Desktop Tooling
    - Build Tooling
--

### What Changes?

- No sub folder checkout
- Commands change (add, commit, checkout, push, pull)
- Limited access control
- Commits are more explicit
- No more copies
- [Git for SVN Users Cheatsheet](https://www.git-tower.com/blog/git-for-subversion-users-cheat-sheet)
